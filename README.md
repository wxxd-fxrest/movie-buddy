## Movie-Buddy
React-Native + Firebase + Api (01)

> 주요기능

+ 한 달 목표치 설정(영화 관람)
+ 관람한 영화에 대한 개인적인 리뷰 및 평점 기록
+ 내가 관람한 영화 리스트 생성
+ 회원가입 / 로그인
  소셜 로그인
  회원 정보 수정
